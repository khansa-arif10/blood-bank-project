## рҹҺ¬ Typical Workflow

### Morning Routine:
1. **Login** вҶ’ Popup appears (if critical)
2. **Read critical blood groups** вҶ’ Note which ones need urgent attention
3. **Click "View Alert Details"** вҶ’ See full inventory status
4. **Take action**:
   - Contact donors for critical blood groups
   - Organize urgent donation camp
   - Check pending requests for those blood groups

### Throughout the Day:
5. **Monitor red badge** вҶ’ Always visible as reminder
6. **Record donations** вҶ’ Badge updates automatically
7. **Click badge anytime** вҶ’ Check current status
8. **Watch badge count decrease** вҶ’ Progress indicator

### End of Shift:
9. **Check if badge still visible** вҶ’ Update next shift
10. **Logout** вҶ’ Alert state resets

### Next Login:
11. **Popup appears again if still critical** вҶ’ Ongoing reminder
12. **Continue working to resolve** вҶ’ Until all blood вүҘ 3 units

## вқ“ Common Questions

**Q: Why does the popup appear every time I login?**
A: It's a reminder that critical action is still needed. It will stop appearing once all blood groups have вүҘ 3 units.

**Q: Can I turn off the popup?**
A: Yes, click "Dismiss". The red badge will remain as a visual reminder.

**Q: What if I want to see the alerts again after dismissing?**
A: Click the red `рҹ”ҙ X URGENT` badge in the navbar, or click the "Alerts" card on the dashboard.

**Q: When does the badge disappear?**
A: When all blood groups have at least 3 units in inventory.

**Q: Does the badge update automatically?**
A: Yes! After recording donations, the badge updates immediately to reflect new inventory levels.

**Q: What happens when I logout?**
A: The dismissal state resets. Next login, the popup will appear again if there are still critical alerts.
